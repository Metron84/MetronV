# METRON V 99

Metron Ventures website - Version 99

## Files

- `index.html` - Main Metron Ventures homepage
- `intelligence.html` - Metron Intelligence R&D services page

## Features

### Homepage (index.html)
- Hero section with quadrant navigation (Ventures, Intelligence, Framework, Tools)
- Quadrant grid with circular orbit boundary
- Three orbiting hotspot buttons with satirical instructions
- Ventures list with completion percentages
- Framework section
- Tools section
- Contact section

### Intelligence Page (intelligence.html)
- Strategic R&D services
- Case studies (Real Estate, Retail, Oil & Gas)
- Research methodology
- Contact CTA

## Design System

- **Colors**: Black (#1a1a1a), White (#fafafa), Light Gray (#e5e5e5), Gold Accent (#d4a03a)
- **Typography**: Helvetica Neue
- **Layout**: Max-width 600px for body copy, responsive design

## Navigation

- Logo in top-left corner links to homepage
- Intelligence page links back to homepage via logo

## Browser Support

Modern browsers (Chrome, Firefox, Safari, Edge)

## Deployment

Ready for GitHub Pages or any static hosting service.
